Organograma Dinâmico

Este organograma dinâmico permite carregar, visualizar e pesquisar a estrutura hierárquica da organização com base em dados de um arquivo Excel. Utilize os controles abaixo para carregar, pesquisar e expandir ou reduzir os níveis hierárquicos conforme necessário.

Para exportar o organograma para PDF, use o botão "Exportar para PDF ou Ctrl+P".
